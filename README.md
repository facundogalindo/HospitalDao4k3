# turnero-medico
Sistema de turnos médicos para atención médica en consultorios, centros de salud o clínicas de tamaño medio, ofreciendo una herramienta integral que facilite la administración de pacientes, médicos y especialidades.

Correr con PY 3.11.X
pip 

Correr backend con uvicorn app.main:app --reload --lifespan off
(Instalar todo lo necesario con pip install -r requirements.txt)    

Correr front con npm start
(Instalar primero esto: npm install)
